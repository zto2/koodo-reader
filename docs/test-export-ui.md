# 导出UI修复测试清单

## 修复内容总结

### 1. UI层级和遮挡问题修复 ✅
- **问题**：统一导出对话框(z-index: 1000)遮挡更多操作菜单(z-index: 9)
- **修复**：
  - 统一导出对话框 z-index: 1000 → 15
  - 更多操作菜单 z-index: 9 → 20
  - 主菜单 z-index: 9 → 10
  - 增加鼠标移动延迟时间：150ms → 200ms

### 2. 功能重复问题修复 ✅
- **问题**：更多操作菜单中存在重复的导出功能
- **修复**：
  - 移除更多操作菜单中的重复导出选项：
    - Export notes
    - Export highlights  
    - Export notes to Anki
    - Export highlights to Anki
    - Export notes to PDF
    - Export highlights to PDF
  - 移除主菜单中的"更多操作"选项
  - 只保留统一导出功能，避免用户困惑

### 3. 响应式设计问题修复 ✅
- **问题**：小屏幕时对话框超出边界
- **修复**：
  - 添加边界检测和位置计算
  - 移动端(≤768px)居中显示
  - 动态调整对话框尺寸
  - 添加窗口大小变化监听器
  - CSS响应式优化：
    - 小屏幕：max-width: calc(100vw - 40px)
    - 超小屏幕：max-width: calc(100vw - 20px)

## 测试步骤

### 基础功能测试
1. 打开应用：http://localhost:3000
2. 导入一本书籍
3. 添加一些笔记和高亮
4. 右键点击书籍，选择导出选项

### UI层级测试
1. 悬停在"导出"选项上
2. 验证统一导出对话框正确显示
3. 验证没有被其他菜单遮挡
4. 测试鼠标在菜单间移动时的稳定性

### 功能整合测试
1. 验证只有统一导出选项可用
2. 确认没有重复的导出功能
3. 测试各种导出格式：PDF、Anki、CSV、Flomo

### 响应式测试
1. 调整浏览器窗口大小
2. 测试小屏幕(≤768px)时的显示效果
3. 验证对话框始终在可视区域内
4. 测试窗口边界情况
